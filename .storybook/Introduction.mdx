# 🚀 Welcome to Your Component Library

This Storybook contains interactive demos of your custom components.  
You can explore, play with inputs, and see how they behave in real-time. 🎨✨  

---

## 🌟 Available Components

- 📋 **DataTable** → Interactive table with sorting, row selection, and loading states.
- 📝 **InputField** → Colorful input box with variants, validation, and helper text.

---

## 🎮 How to Play
- Switch between **stories** using the left navigation.  
- Use the **Controls** panel to tweak values like `placeholder`, `variant`, and `size`.  
- Try the **Playground demos** for a realistic mini frontend experience.
